
# PySPOT

Welcome to PySPOT. A repository dedicated to building an easy-to-use Python
wrapper around [CSPOT](https://github.com/MAYHEM-Lab/cspot), a Functions as a
Service platform for IoT. 

However, this **repository isn't limited to only Python**. This repository also is
for any other wrappers / tools that we make for CSPOT. (Such as a C++ set of
tools / wrappers )

- Check out `pyspot` for the Python wrapper.
- Check out `directory_coming_soon` for the CPP wrapper. 

> Currently, many tools and interfaces to CSPOT are scattered across many
> projects. This repo is simply to gather them together in one place. 

## Get started

To download / install cspot, simply run
``` sh
./install_cspot_bin.sh
```
This will create a `bin` directory and download cspot binaries to it. It will
also add the directory to PATH.
